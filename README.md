# Supply Chain & Logistics Analytics Dashboard | Power BI

## Project Overview

Developed an interactive Power BI dashboard to analyze supply chain performance, supplier efficiency, transportation costs, profitability, and operational KPIs.

---

## Tools Used

- Power BI
- Power Query
- DAX
- Data Visualization
- Business Analytics

---

## Dashboard Features

- KPI Cards
- Dynamic Dashboard Title
- Profit Margin DAX Measure
- Conditional Formatting
- Interactive Slicers
- Drillthrough Navigation
- Supplier Deep Dive Page

---

## Key Metrics Tracked

- Revenue
- Profit
- Profit Margin %
- Lead Time
- Shipping Cost
- Location-wise Profitability
- Supplier Performance

---

## Key Insights

- Skincare products generated highest revenue.
- Supplier 3 recorded the longest lead time.
- Road transportation contributed highest shipping cost.
- Mumbai showed strongest profitability.

---

## Business Recommendations

- Reduce dependency on high lead-time suppliers.
- Optimize transportation expenses.
- Focus inventory planning on high-revenue products.
- Improve operational efficiency for low-profit regions.
